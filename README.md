# 📸 Modern Image Gallery App

A clean and modern **Image Gallery Web App** built using **HTML, CSS, and JavaScript**. This project showcases images in a responsive grid with smooth hover effects and a lightbox preview feature.

---

## 🚀 Features

* 📱 **Responsive Design** – Works on all screen sizes
* 🖼️ **Grid Layout Gallery** – Clean and organized image display
* ✨ **Hover Effects** – Zoom-in animation with captions
* 🔍 **Lightbox View** – Click any image to view in full screen
* 🎨 **Modern UI** – Gradient background with minimal design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid + Animations)
* JavaScript (DOM Manipulation)
